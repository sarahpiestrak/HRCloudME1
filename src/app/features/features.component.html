Payroll generation is a seamless process. Earnings and deductions are automatically compiled into pay statements for each employee. A comparison report highlights the differences with the previous month making it easy to find the differences.

Payroll automatically includes earnings like basic salary, allowances, bonuses and leave encashments. The payroll also includes automated deductions for loan, advance salary, unpaid leave and more.

Salary transfer file is automatically generated and can be sent to the bank. Pay slips can be emailed or printed.
